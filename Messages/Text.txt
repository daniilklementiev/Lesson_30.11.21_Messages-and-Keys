Сообщения. Клавиатура и мышь.

При получении сообщения  дополнительно передаются параметры
wParam (WORD - 16 bit) - числовая характеристика: кол-во символов, код кнопки 
lParam (LONG - 32 bit) - адрес на доп. параметры (структура/массив)

Клавиатура
wParam - key code : числовая характеристика кнопок, в основном, согласующаяся с ASCII
lParam - битовая композиция, в которой разные группы бит, означающие разные данные, например, биты 16-23 (отсчет от 0)
			передают скан-код

(x >> 16) & (0..0 1111 1111)

------------------------------------------------
Мышь
WM_MOUSEMOVE - движение мыши
lParam - соединение двух 16-битных чисел: координата Х и Y курсора мыши. 
Для их разделения рекомендуются:
GET_X_LPARAM(lParam);
GET_Y_LPARAM(lParam);
эти конструкции учитывают знак числа (в отличии от LOWORD и HIWORD)

WM_LBUTTONDOWN - зажатие левой кнопки
WM_LBUTTONUP - отжатие левой кнопки

------------------------------------------------
Таймер
Системный ресурс, позволяющий:
а) посылать хроно-события с определенной переодичностью
б) переодически вызывать выполнение функции 

Запуск таймера:
SetTimer(
	hWnd,		// окно, к которому привязан таймер
	TIMER_1,	// число, передаваемое таймером как свой идентификатор
	1000,		// период (в мс - 1000 мс = 1 сек)
	NULL		// адрес запускаемой функции
	)			// посылает сообщение WM_TIMER в окно hWnd
